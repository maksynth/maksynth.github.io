About

Mak Synth is a hands-on synthesizer project built using Arduino, custom PCBs, and SuperCollider. It’s designed to teach sound design through physical interaction.

Features

  • 12 physical cantilever keys
  • 6 analog potentiometers for sound shaping
  • Arduino + SuperCollider integration
  • Custom PCB and modular wiring system
  • Monophonic synth engine with ADSR control

Build Process

The synth is designed for modular assembly using header pins, 3D printed parts, and minimal soldering. The goal is accessibility and educational use in workshops.
