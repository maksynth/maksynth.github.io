FILES

  • Step 1
    Download the Source Files

    Grab the firmware for the hardware controller and the digital sound engine files below.

    Download .INO (Arduino) Download .SCD (SuperCollider)
  • Step 2
    Flash the Arduino

    Open the .ino file in the Arduino IDE. Connect your synthesizer hardware via USB, select your board port, and hit upload to flash the controller firmware.

  • Step 3
    Boot the Sound Engine

    Open the .scd file in SuperCollider. Start the local server, check that your serial port configuration matches your Arduino, and run the code block to initialize the synth patch.
